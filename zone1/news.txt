
Welcome to asss (a small subspace server) 1.1.0.

asss is a new subspace server, written from scratch. It has lots of cool
new features and improvements over subgame.

This news.txt will be used for updates about new features and other
important information. If you have any suggestions of any kind, please
let me know.

This server is now connected to the SSC billing server. If you notice
any problems with inter-zone features (features that involve the billing
server), please let me know.


Recent updates:
    * lots of changes were made between 1.0 and 1.1. the most
      significant are: support for modules written in Python, a game
      recorder module, updates to the console chat client, and lots of
      bug fixes.
