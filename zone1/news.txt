
Welcome to asss (a small subspace server) 1.3.5.

asss is a new subspace server, written from scratch. It has lots of cool
new features and improvements over subgame.

This news.txt will be used for updates about new features and other
important information. If you have any suggestions of any kind, please
let me know.

This server is connected to the SSC billing server. If you notice any
problems with inter-zone features (features that involve the billing
server), please let me know.


Recent updates:
    * brand new flags code, fixing lots of bugs, and enabling new flag
      games

    * ?modinfo command, cont 0.39pr1 support

    * more detail at http://sscx.net/asss/changelog.txt
